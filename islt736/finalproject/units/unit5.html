Study Guide

  • Be able to analyze word problems to write 2-step equations
  • Be able to interpret the quantities in an equation
  • Be able to use models to solve 2-step equations
  • Be able to solve pequations using the distributive property
  • Be able to Graph the solution of an inequality on a number line
  • Be able to solve an inequality using addition, subtraction, multiplication, and division
  • Be able to write and solve a 2-step inequality
  • Be able to solve multi-step inequalities (more than 2-step)
